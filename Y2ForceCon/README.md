# Y2ForceCon

Legacy force-control package retained for reference.

## Purpose

Contains the original admittance, FAAC, and RL/NAF force-control implementations
used as references while creating `gforce_control`.

## Distribution Note

The current `Y2RobMotion` integration does not directly depend on this package
for force control. It is included to preserve the original implementations and
for comparison/debugging.
