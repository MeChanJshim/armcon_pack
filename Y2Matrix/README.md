# Y2Matrix

Matrix and rotation utility library used by the ArmCon Pack control packages.

## Purpose

Provides matrix operations, homogeneous transforms, spatial-angle conversions,
and rotation utilities used by kinematics, trajectory, and motion-control code.

## Used By

- `Y2Kinematics`
- `Y2Trajectory`
- `Y2RobMotion`
- `Y2FT_AQ`
