# Y2FT_AQ

Force/torque acquisition utility package.

## Purpose

Provides ROS 2 components for force/torque sensor data acquisition, processing,
and publication.

## Dependencies

- `Y2Filters`
- `Y2Matrix`
- `rclcpp`
- `std_msgs`
- `geometry_msgs`
- `std_srvs`
